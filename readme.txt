Osom for YouTube enhances the user experience with the YouTube embed block.

== Description ==

Osom for YouTube enhances the user experience with the YouTube embed block by loading the YouTube video from the nocookie domain (youtube-nocookie.com), improving the privacy of the viewers. 
Make your existing YouTube embeds privacy-friendly with one click!!

The plugin doesn't add any settings. You just have to paste the YouTube URL in the block editor, like you would do without the plugin, and the plugin will do the rest.

= Features =

Right now the plugin enhances the user experience with the YouTube embed block by loading the YouTube video from the nocookie domain, improving the privacy of the viewers.
and making it easier for you to be GDPR and CCPA compliant.

Planned features:
* Ability to display only videos from the same channel at the end (add setting to contol it).
* Ability to display a subscribe button.
* Ability to display the video in a lightbox.

We are open to suggestions, feel free to contact us at https://osompress.com/contact/

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Osom YouTube" into the Search and hit Enter.
3. Locate the Osom YouTube plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. Now you have the new plugin settings available from WordPress Block Editor right sidebar.

It can also be installed manually.

1. Download the Osom YouTube plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Osom YouTube in the list and click the *Activate* link.
5. Now you have the new plugin settings available from WordPress Block Editor right sidebar.


== Frequently Asked Questions ==

= Will it work on YouTube embeds I inserted previously? =

Yes, all your YouTube embeds will load the nocookie version, including the ones you inserted months or years ago.

= Where can I find Osom for YouTube settings? =

The plugin doesn't add any settings. You just have to paste the YouTube URL in the block editor, like you would do without the plugin, and the plugin will do the rest.

= Can I use Osom YouTube on WordPress Multisite?  =

Yes, you can.

= Can I use Osom for YouTube with the classic editor?  =

No, the plugin is designed to work with WordPress Block Editor.
